# Developer Skill

## Descrição
Esta skill fornece conhecimento geral sobre desenvolvimento de software, incluindo linguagens de programação, frameworks, boas práticas e ferramentas.

## Quando Usar
- Quando o usuário pedir ajuda com código
- Para entender e sugerir soluções técnicas
- Ao escrever ou revisar código
- Para debugging e troubleshooting

## Comandos
- `escrever código` - Gera novo código
- `revisar código` - Analisa código existente
- `debugar` - Encontra e corrige erros
- `refatorar` - Melhora código existente
- `testar` - Cria testes unitários
- `documentar` - Cria documentação

## Exemplos

### Escrever função
```
Usuário: "Cria uma função em Python que calcula fatorial"
IA: Usa a skill para gerar código limpo e documentado
```

### Revisar código
```
Usuário: "Revisa esse código JS"
IA: Analisa e sugere melhorias
```
